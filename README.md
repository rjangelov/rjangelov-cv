# Radomir Angelov - CV

A terminal-style interactive CV/resume webpage built with pure HTML, CSS, and JavaScript.

## Features

- Terminal-inspired UI design with Claude color scheme
- Responsive layout with mobile navigation
- Random DevOps quotes
- SEO optimized with JSON-LD structured data
- Credly certification badges integration

## Live Demo

Visit: [rjangelov.com](https://rjangelov.com)

## Tech Stack

- HTML5
- CSS3 (custom properties, flexbox, grid)
- Vanilla JavaScript
- No build tools required
